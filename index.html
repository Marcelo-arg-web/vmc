Semana de trabajo

Elegí la semana (fecha) para cargar el link de WOL, sugerir asignados y guardar.

—
Si intentás cambiar de semana sin guardar, el sistema te va a advertir.

Primeros pasos

  1. Ir a Configuración y pegar tu Firebase Config.
  2. Ir a Login e iniciar sesión con un usuario de Firebase Auth.
  3. Ir a Personas para cargar/editar la lista (sexo, rol, aprobados).
  4. Ir a Semana, pegar link WOL, Cargar, Sugerir, Guardar.
  5. Ir a Tablero para imprimir o exportar PNG para WhatsApp.
Nota: Para leer WOL desde GitHub Pages usamos un “mirror” (r.jina.ai). Si un día falla, en Configuración podés poner un Proxy (Worker) propio.
Consejo: guardá la semana antes de pasar a otra. Si ves “No guardado”, no cambies de semana hasta guardar.
